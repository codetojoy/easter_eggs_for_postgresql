
### Usage

* requires `psql`
    * one option is to install via Home Brew. [see here](https://stackoverflow.com/a/49689589/12704)
* at terminal: 
    * `cp setvars.sh my_setvars.sh`
    * edit `my_setvars.sh` with values
    * `chmod 744 my_setvars.sh`
    * `. ./my_setvars.sh`
* see `go.sh` for full flow
